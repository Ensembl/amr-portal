# Accessing AMR data from the AMR portal

1\. You can explore antimicrobial resistance data using the Query AMR data.

![The entry page](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/2030e0d5-aa70-4f73-8283-7ce77a62f168/ascreenshot.jpeg?tl_px=0,104&br_px=2072,1263&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=167,277)

2\. The antimicrobial resistance portal contains three main data resources that you can select from to begin extracting data.

First select the data resource you would like to use.

- Phenotypic AMR (from CABBAGE antibiograms catalogue)
- Genotype predicted AMR (predictions of AMR resistance from computational methods)
- Phenotypic / genotype AMR availability (Overall data availability)

The screenshot examples below use the Phenotypic AMR resource.

In the "Filter by" header, you can select any of the descriptors to begin subsetting the data. Click on any check box to begin filtering.

![Selecting a first antibiotic](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/b8569bf2-a11d-44f5-8dd3-cbe992ed7c9a/ascreenshot.jpeg?tl_px=0,0&br_px=2072,1158&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=605,168)

3\. You may select one or more antibiotics, in the screenshot example below, azithromycin was selected. You can then select other descriptors in the "Filter by" header, such as Genus or Species.

![Selecting species](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/ebec2cc1-47e2-4715-8dfb-da9c332a3c74/ascreenshot.jpeg?tl_px=0,0&br_px=2072,1158&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=512,99)

4\. Tick any of the options to further subset the results table displayed on the bottom half of the screen. You can scroll through available checkboxes by using the scroll bar under the list of checkboxes.

![Scrolling through checkboxes](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/73a53e18-eaad-41c8-8766-26f643d09a97/ascreenshot.jpeg?tl_px=0,191&br_px=2072,1350&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=522,319)

5\. The number of results will change when additional filters are added.

![Result numbers update](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/e7188f92-174e-4812-8799-972ed4e5884a/user_cropped_screenshot.webp?tl_px=282,0&br_px=2072,1000&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=889,232)

6\. The results table will display all the available portal information for the given resource, including BioSampleID, identifiers, and country information. The table has many columns which are off screen, but can be scrolled to with the scroll bar on the bottom of the results table.

![Scroll through results](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/c3ad284d-95d7-40f9-a214-9f7774f8d2d4/user_cropped_screenshot.webp?tl_px=0,0&br_px=2720,688&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=151,220)

7\. You may continue to add filters depending on the data you would like to extract.

![Adding a resistance phenotype filter](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/75ddc565-bd37-4176-aaf6-657311300a92/ascreenshot.jpeg?tl_px=0,0&br_px=2072,1158&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=658,16)

8\. In this example, a Resistance phenotype is selected: "resistant"

![Adding the resistance phenotype resistant](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/ea4bfd19-8696-4839-a879-a0ae9dc39727/ascreenshot.jpeg?tl_px=0,0&br_px=2072,1158&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=369,152)

9\. This example shows the available results for the options of "Antibiotic: azithromycin" + "Species: Escherichia coli" + "Resistance phenotype: Resistant"

![Showing results of the antibiotic azithromycin in Escherichia coli with a resistant phenotype ](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/ac58b079-b4c6-4775-b105-f62758e229d1/ascreenshot.jpeg?tl_px=0,191&br_px=2072,1350&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=909,333)

10\. You can change the number of results displayed per page by using this drop down menu.

![Change the number of results shown per page](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/bbfa3048-926f-4a7c-8b97-f3f68ae020a1/ascreenshot.jpeg?tl_px=0,191&br_px=2072,1350&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=87,340)

11\. You can scroll through the results table by using the scroll bar on the right hand side.

![Scrolling results](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/800a0660-f51e-4c67-8041-616b530965d0/ascreenshot.jpeg?tl_px=0,50&br_px=2072,1209&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=962,277)

12\. You can download the results by clicking on the download arrow on the right hand side of the results table.

![Downloading results](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/577a9b70-2452-46bf-9e68-63a3647e1085/ascreenshot.jpeg?tl_px=0,153&br_px=2072,1312&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=1003,277)

13\. And then selecting the green download button, which will export the table to your computer in comma separate value format.

![Exporting to CSV](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/33d626c8-f7d8-49db-b5b2-d06ce3c4b2d2/ascreenshot.jpeg?tl_px=0,57&br_px=2072,1216&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=781,277)

14\. You can repeat these steps for any set of filters or descriptions. Each data resource will have its own set of filtering criteria.

![Repeating for other data sets](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/5264c97b-b76f-4054-9229-2af5f7bb6691/ascreenshot.jpeg?tl_px=0,0&br_px=2072,1158&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=160,191)

15\. If you swap to another, you will notice that the filter criteria change to fit this resource.

![Changing criteria](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/936c5ac4-9f60-46fb-a61c-34aef54814cd/ascreenshot.jpeg?tl_px=0,0&br_px=2072,1158&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=161,225)

16\. Each resource has its own set of filter criteria to select from.

![Each resource has its own filters](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/277197a5-312c-4faf-89a8-87b49769c652/ascreenshot.jpeg?tl_px=0,0&br_px=2072,1158&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=135,145)

17\. If you would like to clear your results click on the blue recycle bin icon on the right hand side of the results table.

![Clearing results by clicking on the recycle bin](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-10-30/c5458699-cc99-4abf-a85e-12620e07b1c9/ascreenshot.jpeg?tl_px=0,191&br_px=2072,1350&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=1006,469)

18\. You can clear all selected filters by clicking the green button, but note, it will remove any configuration of the table.
